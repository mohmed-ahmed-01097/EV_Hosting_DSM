# Phase 2 - Behavior-Driven Load Model

Implemented MATLAB modules:

- `simulate_occupancy.m`
- `generate_activities.m`
- `trigger_appliances.m`
- `run_appliance_profile.m`
- `discretize_runs_to_power.m`
- `hvac_power_model.m`
- `ev_model.m`
- `extract_flexibility.m`
- `simulate_household.m`
- `simulate_population.m`

The implementation uses the Phase 0 survey workbook and Phase 1 household assignment to produce household-level daily load profiles with occupancy, activity, appliance, HVAC, controllable-load, and EV metadata.
